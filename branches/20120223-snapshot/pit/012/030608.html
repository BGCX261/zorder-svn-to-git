2003/06/08

頭有點昏啊今天....

早上花了快兩個小時解決個使用者反應的登入問題, 到最後才發現.... 是沒有VBScript 支援!! O_O 救人也~ 不過....總之先把登入時的小窗改寫一番, 搞成如果沒VBScript 的話就直接顯示錯誤, 另外也開始寫問題與解決法的文件了.

下午....想找看有沒有可以直接下載安裝的VBScript包, 這樣子就可以擺著讓人裝了....然而~ 沒有.

不過運氣不錯~ 把IE5 還有IE6 內的VBScript 的cab 檔copy出來, 接著硬把它寫成了AvtiveX object內嵌~ 哇哈哈~~~ 把它變成自動安裝的了~ 接著拿VM內的95裝上IE5 試, 安全性開到最高都還是會自動裝, 頂多出來個問的視窗, 真的是太美滿了~

接著一不作二不休把Flash 是否安裝的訊息也作了, 現在VBS 還有FLASH6 的問題等於是解決了. 接著把這些FAQ 打了打, 喔耶~

晚上又在看是否有Audio Chat的解決方案, 搜了半天都還是很general 的東東, 最後又去MacroMedia逛, 抓了Flash Communication Server 來玩~ 好毒! 硬是要賣錢嘛! O_O 原來啊, 果然一定要FCS 才可以傳視訊與音訊, 在網上再找半天, 並沒有opensource的server release 出來或有解出的協定的文件. 它的ActionScript 又定死成沒法子直接兩個client互傳....所以, 看來我想直接搞flash 語音聊天還是有段路要走.....O_O

FCS 的dev.版上限是五個connection, 幾乎是啥都不能玩.

測夠時已經深夜一點, 只好乖乖去躺了....
