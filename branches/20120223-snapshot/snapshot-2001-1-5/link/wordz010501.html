2001/05/01

五月的第一天。

睡到11點才起床。糟糕，懶散的心越發擴散。今天應該要多努力些。

今天作了決策樹粹取出分類法則的部份。一下午作了粹取程式， 也弄了一組新的實驗圖組。新的實驗圖組的分類正確率為 73%、FDT 時正確率則在 75%。

晚上把非FUZZY 部份的粹出程式與驗證程式寫完，作了一下實驗。對於 Set 4來說， 正確率為 74%，與原先的ID3 EX的 78%正確率相差不多。在Set 3上， 正確率由 70%降到 67%；在Set 2上，正確率由 88%降到 86%。整體說來， 在粹出並拋棄一半的分類決策後正確率都會略降，但是幅度並不大。

接著作粹出前1/3分類決策的對照： Set 2之正確率為88% 降至 85%； Set 3之正確率為70% 降至 68%； Set 4之正確率為78% 降至 75%。

接著作粹出前1/4分類決策的對照： Set 2之正確率為88% 降至 86%； Set 3之正確率為70% 不變； Set 4之正確率為78% 降至 75%。

接著作粹出前1/5分類決策的對照： Set 2之正確率為88% 降至 86%； Set 3之正確率為70% 不變； Set 4之正確率為78% 降至 72%。

除了Set 4以外，其它的圖組在取的分類法則越精簡的狀況下， 分類效果越接近甚或超過ID3 EX。

粹出前1/10分類決策時： Set 2之正確率為88% 降至 86%； Set 3之正確率為70% 降至 61%； Set 4之正確率為78% 降至 72%。

剛補作了Set 5的其它實驗，數據是1/2 → 71%、1/3 → 70%、1/5 → 70%、 1/10 → 72%。
